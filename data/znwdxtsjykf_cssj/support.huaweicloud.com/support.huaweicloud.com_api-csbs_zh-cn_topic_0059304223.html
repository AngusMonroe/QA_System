功能描述

创建备份策略

URI

  • URI

  • 参数说明

    名称

    是否必选

    类型

    说明

    project_id

    是

    String

    -

请求

  • 要素说明

    名称

    是否必选

    类型

    说明

    policy

    是

    policy_create

    创建参数
  • policy_create 字段数据结构说明

    名称

    是否必选

    类型

    说明

    description

    否

    String

    备份策略描述，长度限制：0- 255，除了“<”、“>”不能输入，其它字符均可。

    name

    是

    String

    备份策略名称，长度限制：1- 255，只能由中文、字母、数字、“_”、“-”组成。

    parameters

    是

    policy_param

    备份参数

    provider_id

    是

    String

    备份提供商id

    resources

    是

    List<resource>

    备份对象列表，列表可以为空

    scheduled_operations

    是

    List<scheduled_operation_create>

    调度周期，一个备份策略有且只有一个备份周期，0个或1个复制周期。
  • policy_param 字段数据结构说明

    名称

    是否必选

    类型

    说明

    common

    否

    common_param

    备份策略通用参数
  • resource 字段数据结构说明

    名称

    是否必选

    类型

    说明

    id

    是

    String

    待备份对象id

    type

    是

    String

    备份对象的实体对象类型虚拟机：OS::Nova::Server

    name

    是

    String

    备份对象名称
  • scheduled_operation_create 字段数据结构说明

    名称

    是否必选

    类型

    说明

    description

    否

    String

    调度周期描述，长度限制：0- 255，除了“<”、“>”不能输入，其它字符均可。

    enabled

    否

    Boolean

    是否启用该调度周期，默认为true 不启用时不自动调度，可以手工调度

    name

    否

    String

    调度周期名称，长度限制：1- 255，只能由中文、字母、数字、“_”、“-”组成。

    operation_type

    是

    String

    操作类型，备份或者复制

    Enum:[ backup, copy]

    operation_definition

    是

    operation_definition

    调度周期参数

    trigger

    是

    trigger

    调度策略
  • common_param 字段数据结构说明

    名称

    是否必选

    类型

    说明

    expired_at

    否

    String

    备份策略过期时间（UTC时间，%Y-%m-%dT%H:%M:%S格式）过期之后不能备份、复制，可以恢复、删除默认为不限制

    az

    否

    String

    备份策略中资源的AZ，只能由字母、数字、“_”、“-”、“.”组成
  • operation_definition 字段数据结构说明

    名称

    是否必选

    类型

    说明

    max_backups

    否

    Integer

    单个备份对象自动备份的最大副本数，最大值为99999。-1代表不按副本数清理。

    retention_duration_days

    否

    Integer

    备份副本保留时长，单位天。最长支持99999天。-1代表不按时间清理。

    permanent

    否

    Boolean

    是否永久保留。
  • trigger 字段数据结构说明

    名称

    是否必选

    类型

    说明

    properties

    是

    trigger_properties

    调度器属性
  • trigger_properties 字段数据结构说明

    名称

    是否必选

    类型

    说明

    pattern

    是

    String

    调度器的调度策略，长度限制为10240个字符，参照iCalendar RFC 2445规范，但仅支持FREQ、BYDAY、BYHOUR、BYMINUTE四个参数，其中FREQ仅支持WEEKLY，BYDAY支持一周七天（MO、TU、WE、TH、FR、SA、SU），BYHOUR支持0-23小时，BYMINUTE支持0-59分钟，并且间隔不能小于一小时，一天最大24个时间点。复制策略建议一天只设置一个时间点。
  • 请求样例
POST /v1/{project_id}/policies
{
  "policy" : {
    "name" : "my-plan",
    "description" : "My plan",
    "provider_id" : "c714180d-ea34-4b13-9a5e-577c7c416eec",
    "parameters" : {
      "common" : {
        "expired_at" : "2016-12-31T18:00:00"
      }
    },
    "scheduled_operations" : [ {
      "name" : "my-backup-policy",
      "description" : "My backup policy",
      "enabled" : true,
      "operation_definition" : {
        "max_backups" : 20
      },
      "trigger" : {
        "properties" : {
          "pattern" : "BEGIN:VCALENDAR\r\nBEGIN:VEVENT\r\nRRULE:FREQ=WEEKLY;BYDAY=TH;BYHOUR=12;BYMINUTE=27\r\nEND:VEVENT\r\nEND:VCALENDAR\r\n"
        }
      },
      "operation_type" : "backup"
    } ],
    "resources" : [ {
      "id" : "45baf976-c20a-4894-a7c3-c94b7376bf55",
      "type" : "OS::Nova::Server",
      "name" : "resource1"
    }, {
      "id" : "5aa119a8-d25b-45a7-8d1b-88e127885635",
      "type" : "OS::Nova::Server",
      "name" : "resource2"
    } ]
  }
}
