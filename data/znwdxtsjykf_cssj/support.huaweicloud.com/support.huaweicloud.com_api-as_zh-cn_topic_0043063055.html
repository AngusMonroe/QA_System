创建弹性伸缩配置

功能介绍

创建弹性伸缩配置。伸缩配置定义了用于创建弹性伸缩组中实例的配置。AutoScaling为某个伸缩组自动添加实例时，会根据配置创建实例。

URI

  • URI格式

    POST /autoscaling-api/v1/{tenant_id}/scaling_configuration

  • 参数说明

    名称

    是否必选

    参数类型

    说明

    tenant_id

    是

    String

    租户标识。

请求

  • 要素说明

    名称

    是否必选

    参数类型

    说明

    scaling_configuration_name

    是

    String

    伸缩配置名称(1-64个字符)，只能包含中文、字母、数字、下划线或中划线。

    instance_config

    是

    字典数据结构

    实例配置信息。
  • instance_config字段数据结构说明
    说明：
    • 目前，华北区暂时不支持Cloud-init，可以使用key_name或adminPass参数。
    • 使用支持Cloud-init或Cloudbase-init功能的镜像创建云服务器时，只能使用key_name参数，adminPass参数无效。对于Linux弹性云服务器，如果需要注入密码，只能使用userdata进行注入，对于Windows弹性云服务器，如果需要注入密码，只能通过元数据admin_pass进行注入。
    • 使用不支持Cloud-init或Cloudbase-init功能的镜像创建云服务器时，adminPass和key_name参数都无效，请使用镜像内置的密码或证书登录。

    名称

    是否必选

    参数类型

    说明

    instance_id

    否

    String

    云服务器ID，当使用已存在的云服务器的规格为模板创建弹性伸缩配置时传入该字段，此时flavorRef、imageRef、disk字段不生效。

    当不传入instance_id字段时flavorRef、imageRef、disk字段为必选。

    flavorRef

    否

    String

    云服务器的规格ID，定义了云服务器CPU、内存大小。通过查询弹性云服务云服务器规格详情和扩展信息列表接口获取，详见《弹性云服务器接口参考》的“查询云服务器规格详情和扩展信息列表”。

    imageRef

    否

    String

    镜像ID，同image_id，通过查询镜像服务镜像列表接口获取，详见《镜像服务接口参考》的“查询镜像列表”。

    disk

    否

    列表数据结构

    磁盘组信息，系统盘必选，数据盘可选。

    key_name

    否

    String

    登录云服务器的SSH密钥名称，与adminPass互斥，且必选一个。

    Windows弹性云服务器不支持使用密钥登陆方式。

    说明：

    此参数仅适用于华北区用户。

    key_name

    否

    String

    登录云服务器的SSH密钥名称。

    说明：

    此参数适用于华南、华东、东北区用户。

    adminPass

    否

    String

    云服务器管理员账户初始登录密码，Linux管理员账户为root，Windows管理员账户为Administrator。该字段与key_name字段互斥，且必选一个。

    密码复杂度要求：

    • 长度为8-26位。
    • 密码只能包含大写字母、小写字母、数字和特殊字符（!@$%^-_=+[{}]:,./?）且至少包含四种字符中的三种。
    • 密码不能包含用户名或用户名的逆序。
    • Windows系统密码不能包含用户名或用户名的逆序，不能包含用户名中超过两个连续字符的部分。
    说明：

    此参数仅适用于华北区用户。

    personality

    否

    列表数据结构

    注入文件信息。仅支持注入文本文件，最大支持注入5个文件，每个文件最大1KB。

    public_ip

    否

    字典数据结构

    配置云服务器的弹性IP信息，弹性IP有两种配置方式。

    • 不使用（无该字段）
    • 自动分配

    user_data

    否

    String

    cloud-init用户数据。

    支持注入文本、文本文件或gzip文件。文件内容需要进行base64格式编码，最大长度32KB。

    说明：

    此参数暂时不适用于华北区用户。

    metadata

    否

    字典数据结构

    创建云服务器元数据。
  • disk字段数据结构说明

    名称

    是否必选

    参数类型

    说明

    size

    是

    Integer

    磁盘大小，容量单位为GB。

    系统盘输入大小范围为40~32768，且不小于镜像中系统盘的最小(min_disk属性)值。

    数据盘输入大小范围为10~32768。

    volume_type

    是

    String

    磁盘类型，有SSD、SATA、SAS三种。

    disk_type

    是

    String

    系统盘还是数据盘，DATA表示为数据盘，SYS表示为系统盘。
  • personality字段数据结构说明

    名称

    是否必选

    参数类型

    说明

    path

    是

    String

    注入文件路径信息。

    1. Linux系统请输入注入文件保存路径，例如 “/etc/foo.txt”。
    2. Windows系统注入文件自动保存在C盘根目录，只需要输入保存文件名，例如 “foo”，文件名只能包含字母和数字。

    content

    是

    String

    注入文件内容，需要进行base64格式编码。
  • public_ip字段数据结构说明

    名称

    是否必选

    参数类型

    说明

    eip

    是

    字典数据结构

    配置云服务器自动分配弹性IP时，创建弹性IP的配置参数。
  • eip字段数据结构说明

    名称

    是否必选

    参数类型

    说明

    ip_type

    是

    String

    IP地址类型。

    类型枚举值：

    • 5_bgp：全动态BGP
    • 5_lxbgp：BGP
    • 5_telcom：中国电信
    • 5_union：中国联通

    bandwidth

    是

    字典数据结构

    IP地址带宽参数。
  • bandwidth字段数据结构说明

    名称

    是否必选

    参数类型

    说明

    size

    是

    Integer

    带宽（Mbit/s），按需分配取值范围为1-100。

    share_type

    是

    String

    带宽的共享类型。

    共享类型枚举：PER，表示独享。

    目前只支持独享。

    charging_mode

    是

    String

    带宽的计费类型。

    • 字段值为“bandwidth”，表示按带宽计费。
    • 字段值为“traffic”，表示按流量计费。
    • 字段为其它值，会导致创建云服务器失败。
  • metadata字段数据结构说明

    名称

    是否必选

    参数类型

    说明

    用户自定义字段键值对

    否

    String

    用户自定义数据总长度不大于512字节。用户自定义键不能是空串，不能包含符号.，以及不能以符号$开头。
  • 请求样例
{
    "scaling_configuration_name": "as-config-tlzq",
    "instance_config": {
        "flavorRef": "103",
        "imageRef": "627a1223-2ca3-46a7-8d5f-7aef22c74ee6",
        "disk": [
            {
                "size": 40,
                "volume_type": "SATA",
                "disk_type": "SYS"
            }
        ],
        "key_name": "100vm_key"
    }
}

响应

  • 要素说明

    名称

    参数类型

    说明

    scaling_configuration_id

    String

    伸缩配置ID。
  • 响应样例
{
    "scaling_configuration_id": "f8327883-6a07-4497-9a61-68c03e8e72a2"
}

返回值

  • 正常

    200

  • 异常

    返回值

    说明

    400 Bad Request

    服务器未能处理请求。

    401 Unauthorized

    被请求的页面需要用户名和密码。

    403 Forbidden

    对被请求的页面访问禁止。

    404 Not Found

    服务器无法找到被请求的页面。

    405 Method Not Allowed

    请求中指定的方法不被允许。

    406 Not Acceptable

    服务器生成的响应无法被客户端所接受。

    407 Proxy Authentication Required

    用户必须首先使用代理服务器进行验证，这样请求才会被处理。

    408 Request Timeout

    请求超出了服务器的等待时间。

    409 Conflict

    由于冲突，请求无法被完成。

    500 Internal Server Error

    请求未完成。服务异常。

    501 Not Implemented

    请求未完成。服务器不支持所请求的功能。

    502 Bad Gateway

    请求未完成。服务器从上游服务器收到一个无效的响应。

    503 Service Unavailable

    请求未完成。系统暂时异常。

    504 Gateway Timeout

    网关超时。
