表2 metadata字段数据结构说明

参数

参数类型

描述

name

string

在一个命名空间里，name必须是唯一的。当创建资源时，即使部分资源允许客户端自动请求一个name，但是该参数为必选。name主要是用于创建幂等配置定义。不支持更新。

uuid

String

一个对象的UUID在时间和空间上都是独特的。它是由创建资源成功的服务生成的，并且不允许使用PUT操作更新。由系统生成，只读。

spaceuuid

String

用于标示用户所在的空间id。

createAt

String

该参数为代表服务创建时间的时间戳。不能保证该值在隔离操作前生效。客户端不允许设置该值。它使用的是RFC3339协议，并且基于UTC。由系统生成，只读。对于列表为NUll。

updateAt

String

该参数表示对象更新时的服务时间戳。不能保证该值在隔离操作前生效。客户端不允许设置该值。它使用的是RFC3339协议，并且基于UTC。由系统生成，只读。对于列表为NUll。
