查询云服务器详情列表

功能介绍

查询云服务器详情信息列表。

URI

  • URI格式

    GET /v2/{project_id}/servers/detail{?changes-since,image,flavor,name,status,host,limit,marker,not-tags}

  • 参数说明

    名称

    是否必选

    说明

    project_id

    是

    租户ID。

以下列表为查询云服务器时可选的查询检索参数

名称

是否必选

参数类型

说明

changes-since

否

String:DateTime

云服务器上次更新状态的时间戳信息。时间戳为UTC格式。

image

否

String

镜像ID。

在使用image作为条件过滤时，不能同时支持其他过滤条件和分页条件。如果同时指定image及其他条件，则以image条件为准；当条件不含image时，接口功能不受限制。

flavor

否

String

云服务器规格ID。

name

否

String

云服务器名称。

status

否

String

云服务器状态。

host

否

String

主机节点名称。

limit

否

Integer

查询返回云服务器数量限制。

marker

否

String

从marker指定的云服务器ID的下一条数据开始查询。

not-tags

否

String

查询tag字段中不包含该值的云服务器。

请求

不涉及

响应
